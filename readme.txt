# RateRide
RateRide is an android app which collects sensor data(Gyroscope, Accelerometer and GPS(including speed)) and user feedback during taking a trip in AppCab.
Installation
## Installation

Using the apk install the app and provide location permission on runtime and for giving storage permission go to settings--> apps --> RateRide --> Permissions --> Storage --> Allow(on)


```bash
After giving both the permission close the app and swipe out from RAM, and its
ready to use.
```

## Files

```python
The app will create 4 files, 

--> myappfeedback.txt
--> Accel.txt
--> Gyroscopedata.txt
--> gpsdata.txt

Each file has timestamped data.
File Path --> /storage/emulated/0/

```
